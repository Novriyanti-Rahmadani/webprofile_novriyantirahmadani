Skill

1

HTML(Basic)

HTML adalah singkatan dari "Hypertext Markup Language" (Bahasa Markup Hiperteks). Ini adalah bahasa pemrograman yang digunakan untuk membuat dan mendesain halaman web. HTML digunakan untuk mengatur struktur dan konten dasar sebuah halaman web, seperti teks, gambar, tautan, dan elemen-elemen lainnya. HTML menggunakan tag atau penanda (markup) untuk mengidentifikasi bagian-bagian berbeda dari halaman web dan memberikan instruksi kepada browser web tentang bagaimana halaman tersebut harus ditampilkan kepada pengguna.

2

CSS (Basic)

CSS adalah singkatan dari "Cascading Style Sheets" (Lembar Gaya Berantai) yang digunakan untuk mengontrol tampilan halaman web dengan mengatur elemen-elemen HTML seperti warna, font, dan tata letak, memisahkan tampilan dari struktur konten dasar HTML. Ini memungkinkan perubahan tampilan tanpa mengubah konten dasar.

3

JavaScript (basic)

JavaScript adalah bahasa pemrograman yang sering digunakan dalam pengembangan web untuk membuat halaman web lebih interaktif dan dinamis. JavaScript tidak sama dengan HTML atau CSS, yang digunakan untuk mengatur tampilan dan struktur halaman web. Sebaliknya, JavaScript memberikan kemampuan untuk mengendalikan perilaku halaman web dan menambahkan fungsi interaktif ke dalamnya.
